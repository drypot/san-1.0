   주의 사항
  ──────

   이 프로그램에 의한 피해에 대해서 보상을 요구할 수 없습니다.

   이 프로그램은 FREEWARE 이며,
   상업적 목적의 배포를 금합니다.
   피치 못할 경우, 제작자에게 미리 연락하시기 바랍니다.


   빨리 시동 부터
  ────────

   sansetup 을 실행시키십시요.

   산이 사용하는 화면 구동기와 환경 설정을 합니다.
   초기 값으로 CGA 부터 VGA 까지 메뉴가 나타납니다. 자신이 사용하는
   화면 출력 기기에 맞추시기 바랍니다.

   화면 구동기의 등록 여부를 묻는 질문이 나타납니다.
   [ENTER] 치십시요.

   '환경저장' 을 선택하십시요.

   '작업마침'을 선택하십시요.

   san 을 실행시키십시요.

   기타 화면 구동기
  ─────────

   svga800.drv 는 아래 열거된 SVGA 를 지원하도록 김헌수님이
   제작 공개해 주신 것입니다. 사용을 원하시면 아래 명령으로 드라이버
   화일을 만들어 주신후 산을 실행시키시면 됩니다.

   드라이버 만들기>  copy svga800.drv sanvideo.drv

   산 1.2.1 판은
  ────────

   큰틀이 운영을 중단한 이후, 제손으로 정리 배포된 산 꾸러미는
   없었습니다. 예상외로, 문의 전화가 많이 왔고, 너무 자료가
   분산되어 있어 사용자들께 어려움이 있는 것 같아, 그동안
   공개했던 산 꾸러미를 정리해서 올려드립니다.

   1.2 판에 대해서 개선된 점은 없습니다.

   1.2.1 판은 Freeware 입니다.


   앞으로의 계획에 대해서
  ────────────

   참고) 이미 산 1.9.0 이 공개됐었습니다. 몇가지 이유로 이 1.9 판은
   2.X.X 로 다시 판수 매김 되어 공개될 것입니다.

   2.X.X 판은 제작자가 같고, 프로그램 이름이 같음에도 불구하고,
   1.2.X 판과 다른 프로그램입니다.

   산 2.X.X 는 지금 시작입니다. 아마도 제자리를 찾는데, 일 년여
   걸릴것으로 예상합니다.

   더이상 1.2.X 판 대해서 기능 개선에 대한 투자는 없을 것입니다.
   1.2.1 판이 제가 공개하는 마지막 1.2.X 판입니다.
   앞으로 전 2.X.X 판의 개발에만 전념하려고 합니다.

   1.2.X 판의 소스 코드를 같이 공개합니다. 이 소스를 사용하거나,
   발전시키는 것은 나머지 여러분의 자유입니다. 단, 이 소스를 이용해서
   수정된 판은 무료로 공개되어야 합니다. 그리고 혼돈을 막기 위해서
   2.0 이상으로 판수를 메김지 마시기 바랍니다.

   나가는 글
  ──────

   1991 년, 고등학교 2 학년 제학중에, 한글 사용이 가능한 에디터를
   '산' 이라 이름짓고 공개하기 시작.

   지금은 건국대학교 전자계산학과 2 학년.
   C 로 작성하던 산 1.2.X 의 제작을 종료하고,
   486 에서 C++ 로 산 2.X.X 를 코딩 중.

   산은 내 인생을 바꾼 프로그램.
   한 사람의 인생을 다른 사람이 완전히 이해할 수 없듯이,
   지금 내가 하는 행동을 이해하는 사람은 하나도 없음.
   부탁이건데, 내 앞에서 내가 하는 행동에 대해서 가치를 가늠하려고
   하지 말기 바람.

   마지막으로,

   난 변덕이 심한 사람임.
